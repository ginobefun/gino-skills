第一条，今天评分最高的内容，来自播客「屠龙之术」。主播庄明浩花了一百分钟，基于七十页 PPT，把二月以来中美 AI 市场的核心变局做了一次系统梳理。

这期播客想回答的问题是：春节前后 AI 行业到底发生了什么？宏观层面该怎么理解这些变化？

先说美国。庄明浩观察到一个很刺激的数字：科技巨头的 AI 基础设施资本支出，已经从百亿美金量级冲向千亿。亚马逊、微软、Google、Meta，每家都在疯狂扩张算力和数据中心。2025 年已经是一个大年了，2026 年还要更大。但问题是，英伟达的财报明明很漂亮，股价却原地踏步。庄明浩分析说，市场已经对"超预期"麻木了，甚至开始担心这些天量投入最终能不能收回来。

然后是 OpenClaw。庄明浩用了一个很好的判断框架：AI 从"对话"到"执行"的工程化跨越。以前的 AI 是你问它答，现在 OpenClaw 获得了你本地系统的权限，能真的替你做事。他引用了一句话我印象很深："龙虾本质上是一个 AI 陪伴产品，这种念想的体感确实跟我们去用一个 chatbot 完全不同。"

最让人思考的是他提到的"2028 末世论"。逻辑链条是这样的：AI 吃掉软件、软件公司裁员、裁员导致消费下降、消费下降影响更多行业，可能形成恶性循环。他说："我们曾经认为中国的 SaaS 会像美国 SaaS 那么值钱，现在看美国的 SaaS 会像中国的 SaaS 那么不值钱。"

中国这边，春节期间模型厂商密集发布新版本，字节 Seedance 2.0 在视频生成领域有了明显突破。另一个焦点是林俊旸离开阿里云的事，背后反映了技术团队追求前沿和大公司追求商业化之间的张力。

说完了宏观分析，我们看一个具体的技术突破。阿里巴巴通义实验室刚刚开源了 Mobile-Agent-v3.5，这是一套真正能跨平台使用的 GUI Agent 基座模型。

为什么说"真正"？因为之前的 GUI Agent 基本都停留在 Demo 阶段，换个平台、换个分辨率就不好使了。通义这次要解决的，正是数据获取难、训练不稳定、跨端不泛化这三个硬骨头。

他们的解法有三个关键创新。第一，混合数据飞轮。用大模型合成复杂的多窗口场景，再通过有向无环图组织工作流，在真实设备上自动回测。简单说，就是让 AI 自己造训练数据、自己验证质量。

第二，统一思维合成。给模型轨迹的每一步都加上结构化思考——屏幕描述、动作反思、关键信息记忆。这让模型在处理长程任务时不会"忘了前面干了什么"。

第三，他们提出了一个叫 MRPO 的强化学习算法，专门解决多平台混合训练时的梯度冲突问题。以前训练手机端的经验会干扰桌面端，现在可以稳定泛化了。

模型从 2B 到 32B 全覆盖，小的用于边端低延迟执行，大的用于深度规划。作者说了一句很到位的话："不是所有任务都需要 32B，但每个场景都需要可部署。"在二十多个基准测试中全部达到了 SOTA。

这意味着 GUI Agent 终于从"能做 Demo"走向了"能上线用"的阶段。对于做自动化、做 RPA 的团队来说，这是一个值得密切关注的开源项目。

今天第三条精讲，是来自「十字路口 Crossing」的一期播客：二十个问题搞懂 OpenClaw。上一条我们聊了宏观视角，这期播客则是从产品和创业的角度做了一次系统拆解。

嘉宾们提出了一个很直观的比喻：ChatGPT 更像一个咨询师，你问它什么它都能侃侃而谈，但它不动手。OpenClaw 更像一个实习生，虽然偶尔犯错，但它真的能给你干活、能出活。

我觉得最有启发的是关于三层记忆系统的分析。OpenClaw 的记忆分三层：最底层是原始日志，中间层是从日志中提炼的中期记忆，最上层是沉淀下来的长期偏好。正因为有了这套记忆机制，它跟你对话的时候会有一种"默契"的感觉。嘉宾说的一句话很有意思："它显得更有人味，不是因为它更聪明，而是因为它有记忆、有默契。"

另一个有趣的观察是关于容错度。人对人天然比对机器宽容。当 OpenClaw 因为拟人化设计让你觉得它是"活"的，你就会更愿意容忍它偶尔犯的错。这不是技术突破，而是产品设计的洞察，但它深刻改变了整个产品的体验。

嘉宾还展望了 Agent 时代的创业方向：第一，把 Agent 集成到微信、飞书这样的 IM 入口，降低使用门槛。第二，为 Agent 构建专用的基础设施，比如身份认证和支付体系。第三，做 ToB 数字员工，通过积累执行轨迹建立护城河。他们还提到一个有意思的观点：开源的 Agent 等于全世界在帮你养孩子，随着 AI Coding 成熟，开发者和使用者的界限会越来越模糊。

好，以上是今天的三条精讲。接下来快速过七条同样值得关注的内容。

第四条，Founder Park 拆解了 Product Hunt 上四十多款 OpenClaw 相关产品，梳理出五大生态维度：Agent 间社交、记忆管理、云托管部署、安全运维、硬件集成。OpenClaw 已经不只是一个产品，它正在长成一条完整的产业链。

第五条，Lenny's Podcast 专访了 Applied Intuition 的 CEO Qasar Younis。这家估值一百五十亿美元的公司你可能没听过，它做的是物理世界的 AI——农业、采矿、建筑自动化。Younis 倡导激进务实主义：隐形模式和工程纪律比公开炒作更重要。

第六条，来自 The Pragmatic Engineer。金融科技公司 Ramp 分享了 AI 产品实战经验：从数千个单一智能体合并为统一对话界面 Omnihat，内部编码智能体已经处理了超过百分之五十的 PR。有意思的是他们的 Policy Agent 能把自然语言的费用政策直接转化成可执行规则。

第七条，Paul Graham 的新文章「品牌时代」，宝玉翻译。核心观点是：设计是追求最优解的向心力，品牌是追求差异化的离心力，两者本质相悖。当产品的实质差异消失，品牌就开始主导一切。但真正的卓越工作应该跟着问题走，而不是跟着品牌走。

<!-- SEGMENT: quick-2 -->
第八条，InfoQ 中文的一篇深度分析：为什么现有的 Agent Infra 无法支撑生产级应用。作者指出现有的 Kubernetes 和 Temporal 都是基于确定性执行的假设，而 Agent 的长程运行和不确定决策需要全新的基础原语：副作用日志、能力隔离和分叉恢复。

第九条，同样来自 InfoQ 中文的 Claude Code 之父 Boris Cherny 访谈。他详述了 Claude Code 从一个脚本演进为全自动 Agent 的过程，放弃 RAG 转向 Agentic Search 是一个关键决策。另外 Anthropic 的组织文化也很有意思：不写 PRD、全员统一"技术参谋"职称、鼓励通才模式。

第十条，硅星人 Pro 发布了二月 AI 产品应用榜。字节豆包借春晚红包 MAU 突破三亿一千五百万，全球排第二。阿里千问靠三十亿补贴月增长百分之五百五十二，冲到全球第三。国内 C 端五强格局基本确立：豆包、千问、夸克、DeepSeek、腾讯元宝。
