## 头条

**1. ChatGPT 冒充律师事件引发千万美元诉讼**

一名女性听信 ChatGPT 建议解雇了自己的真人律师，随后 ChatGPT 为她生成了 40 多份法庭文件，引用的法条、判例和法官全是 AI 编造的。对方律师团队花费 30 万美元回应这些虚构文件，数月后才被发现。OpenAI 面临 1000 万美元诉讼。这一事件再次敲响 AI 幻觉问题的警钟，尤其是在高风险专业领域。[链接](https://x.com/TukiFromKL/status/2030139120937292237)

**2. Claude Code 发布 /loop：AI 编程进入自主循环时代**

Boris Cherny（Claude Code 团队）宣布推出 /loop 功能，允许用户设置最长 3 天的循环任务。典型用例包括自动监控 PR 并修复构建问题、每天早晨通过 Slack MCP 生成消息摘要等。这标志着 AI 编程工具从单次交互迈向持续自主执行。Cherny 还透露 Claude Code 本身已 100% 由 Claude Code 编写。[链接](https://x.com/bcherny/status/2030193932404150413)

**3. Karpathy 开源 autoresearch：用 AI Agent 跑通宵实验**

Andrej Karpathy 将 autoresearch 项目打包为独立仓库并开源。核心理念是人类编写研究策略提示词（.md），AI Agent 自主迭代训练代码（.py），每轮实验固定 5 分钟，一晚可完成约 100 次完整训练。这将 AI 研究的瓶颈从算力和人力转向了提示词工程质量。[链接](https://x.com/karpathy/status/2030371219518931079)
