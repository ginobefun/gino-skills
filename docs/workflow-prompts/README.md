# Workflow 评分提示词

本目录存放 bestblogs.dev 网站 workflow 自动评分系统使用的提示词，与 skills 中的评分标准对齐。

## 文件说明

| 文件 | 用途 | 对应 Skill |
|------|------|-----------|
| `article-analysis-prompt.md` | 文章分析与评分 | bestblogs-process-articles |
| `tweet-analysis-prompt.md` | 推文分析与评分 | bestblogs-process-tweets |

## 评分标准源

所有提示词的评分标准统一基于 `docs/scoring-rubric.md`（BestBlogs 统一评分规则 v2）。

## 与旧版提示词的关键差异

### 文章分析提示词

1. **分类体系扩展**：从 4 个 domain 扩展为 8 个一级分类（新增 Productivity_Growth, Finance_Economy, News_Media, Lifestyle_Culture），二级分类从 4 个扩展到完整枚举
2. **来源权威性机制**：新增 HIGH/MEDIUM/LOW 来源优先级，设定得分天花板（MEDIUM ≤93, LOW ≤89）
3. **原创性评估**：新增 7 级原创性分层（原创首发 → 纯搬运），编译转述类上限 89
4. **评分分布收紧**：95-100 从 <10% 收紧到 <1%，90-94 从 ~20% 收紧到 ~5%，<75 从 ~20% 扩大到 ~35%
5. **评估维度调整**：从信息价值为主调整为内容深度(35)+实用性(25)+相关性(20)+表达质量(10)+创新性(10)
6. **校准检查强化**：新增 4 级校准检查（≥95/≥90/≥85/天花板），每级有明确的必要条件
7. **新增减分项**：教程入门类、领域偏离类、质量问题类、低质聚合类
8. **新增内容提取**：mainPoints（核心观点）和 keyQuotes（金句引用）

### 推文分析提示词

1. **分类体系扩展**：与文章一致，从 4 个扩展为 8 个一级分类
2. **来源权威性机制**：新增来源优先级和天花板
3. **评分分布调整**：90-100 从 <10% 拆分为 95-100(<1%) + 90-94(~8%)，强调推文时效性价值——HIGH 来源重磅首发允许 92-94
4. **评估维度调整**：信息价值权重从 35 提升到 40，原创与深度从 20 降到 15
5. **推文类型基准分**：新增完整的类型基准分表，限定各类型合理区间
6. **新增减分项**：新闻转述、同事件重复报道、领域偏离
7. **极短推文约束**：<50 字推文上限 82（里程碑除外）
8. **防通胀指标**：明确一批推文中 85+ 不超过 20%

## 维护方式

1. 评分标准变更 → 先更新 `docs/scoring-rubric.md`
2. 再同步更新 workflow 提示词和 skill 的 `analysis_rubric.md`
3. 确保三者一致
