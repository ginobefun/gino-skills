# 视频视觉设计指南

## 整体定位

内容视频是将文字/音频内容转化为**视觉增强版**。画面与配音内容同步切换，每 8-15 秒切换一张 slide。视觉的核心目的是**帮助观众更好地理解和记住内容**。

**核心原则**:
- **声画同步**: slides 切换与配音一一对应，观众看到的就是听到的
- **节奏鲜明**: 高信息密度与视觉呼吸交替，不停滞不赶场
- **素材优先**: 优先使用原文配图，不足时 AI 生成
- **简洁克制**: 画面服务于内容，不喧宾夺主
- **无等待感**: 每张 slide 都有信息增量，没有空白等待

## 品牌视觉体系

### 配色

| 角色 | 色值 | RGB | 用途 |
|------|------|-----|------|
| 墨蓝 | `#1a365d` | 26, 54, 93 | 主色、标题、品牌 |
| 琥珀 | `#d97706` | 217, 119, 6 | Top 1 唯一强调色 |
| 米白 | `#fefdfb` | 254, 253, 251 | 背景 |
| 深炭灰 | `#374151` | 55, 65, 81 | 正文文字 |
| 中灰 | `#6b7280` | 107, 114, 128 | 辅助信息 |
| 淡灰分隔 | `#e5e7eb` | 229, 231, 235 | 分隔线 |

### 字体

| 用途 | 字体栈 |
|------|--------|
| 标题/品牌 | Georgia, 'Noto Serif SC', serif |
| 正文/摘要 | -apple-system, 'PingFang SC', 'Noto Sans SC', sans-serif |
| 数字/评分 | 'Inter', 'SF Mono', monospace |

### 视觉气质

高端设计杂志（Monocle、Kinfolk）+ 瑞士国际主义排版:
- 大量留白，信息有呼吸感
- 无渐变、无发光、无粒子效果
- 平面设计，不堆叠阴影
- 克制的动效，以缓入缓出为主

## 视频结构（场景分解）

### 画面尺寸

- **横版**: 1920×1080 (16:9)，默认
- **竖版**: 1080×1920 (9:16)，可选参数

### 通用场景流程

```
[Scene 1] 品牌/标题开场     3-5秒
[Scene 2] 关键词展示        3-4秒
[Scene 3-N] 精讲项 Slides   对应音频时长
[Scene N+1] 速览引导        2-3秒
[Scene N+2] 速览卡片组      对应音频时长
[Scene Last] 品牌结尾       3-5秒
```

场景数量根据内容和模板动态调整。Article 模板无速览段。

## 各场景设计

### Scene 1: 品牌/标题开场（3-5 秒）

显示视频标题（`title` 字段）和日期。可选显示品牌名。

```
┌──────────────────────────────────┐
│                                  │
│        {title}                   │  ← 墨蓝衬线体，居中
│        {date}                    │  ← 中灰，小号
│                                  │
└──────────────────────────────────┘
```

**模板适配**:
- Daily: 标题如 "03.14 BestBlogs 早报"，日期小号
- Weekly: 标题如 "第 11 周精选"，副标题 "03.08 - 03.14"
- Article: 文章标题 + 来源名

### Scene 2: 关键词（3-4 秒）

关键词标签依次淡入（每个间隔 6 帧）。

**模板适配**:
- Daily: 今日 4-6 个热点关键词
- Weekly: 本周 3-4 个主题关键词（对应后续主题段）
- Article: 文章 3-4 个核心标签

### Scene 3-N: 精讲场景 — Slides 模型

每条精讲由 slides 组成，slides 间用 0.8s 交叉淡入淡出（fade）。

#### 6 种 Slide 类型

| 类型 | 布局 | 用途 | 信息密度 |
|------|------|------|---------|
| `cover` | 全幅大图 + 底部暗色渐变 + 标题 | 引入 | 低（视觉吸引） |
| `source-card` | 白色卡片 UI + 评分徽章 | 来源预览 | 中 |
| `problem` | 居中大字 | 核心问题 | 低（制造悬念） |
| `point` | 左图(60%) + 右文(40%) | 观点 | 高（核心信息） |
| `quote` | 斜体居中 + 装饰引号 | 金句 | 低（消化空间） |
| `takeaway` | 暗色叠加 + 白字 | 总结 | 中（收束思考） |

#### 节奏编排原则

slide 类型的排列要形成**信息密度波动**，避免连续高密度或连续低密度：

```
信息密度
 高     ┃         ┃     ┃
 中  ┃  ┃    ┃    ┃  ┃  ┃  ┃
 低  ┃  ┃  ┃ ┃  ┃ ┃  ┃  ┃  ┃
    cover src prob pt  pt  qt  pt  tkwy tkwy
    引入 → 背景 → 悬念 → 观点 → 呼吸 → 深入 → 收束
```

**排列规则**:
- `cover` → `source-card` 或 `problem`（低→中，建立背景）
- `problem` → `point`（低→高，回答问题）
- 连续 `point` 不超过 3 张（之后插入 `quote` 或 `takeaway` 降密度）
- `quote` 后接 `point` 或 `takeaway`（呼吸后继续或收束）
- `takeaway` 放最后 1-2 张（总结收束）

#### 各模板 Slides 编排

**Daily 模板**（精讲 6-10 张/条）:

```
[cover]        12-15%  全幅封面图 + 标题
[source-card]  8-10%   来源预览（可省略非 Top 1）
[problem]      10-12%  居中问题文字
[point]        15-20%  观点 1 + 配图
[point]        12-15%  观点 2 + 配图
[quote]        8-10%   金句引用
[point]        12-15%  观点 3 或补充（可选）
[takeaway]     12-18%  个人思考 + 总结
```

**Weekly 模板**（主题段 8-12 张/段）:

```
[cover]        10-12%  主题封面（用主题词非文章标题）
[point]        12-15%  内容 A 核心观点
[point]        12-15%  内容 A 补充
[problem/quote] 8-10%  过渡到内容 B
[point]        12-15%  内容 B 核心观点
[point]        12-15%  内容 B 与 A 的关联
[quote]        8-10%   金句或洞察
[takeaway]     15-18%  这个主题的整体判断
```

**Article 模板**（12-15 张）:

```
[cover]        8-10%   文章封面 + 标题
[source-card]  6-8%    来源、作者、评分
[problem]      8-10%   核心问题
[point]        10-12%  观点 1 + 配图
[point]        10-12%  观点 2 + 配图
[quote]        6-8%    金句 1
[point]        10-12%  观点 3 + 配图
[point]        8-10%   观点 4（可选）
[quote]        6-8%    金句 2（可选）
[point]        8-10%   延伸思考
[takeaway]     8-10%   个人判断
[takeaway]     8-10%   行动建议
```

#### Top 1 特殊处理

Cover slide 序号使用琥珀色，整体视觉稍强调。

#### Ken Burns 效果

含图片的 slide 应用缓慢放大（1.0 → 1.05），增加画面动感，避免静止感。

#### 无图片 Fallback

- `cover`/`point`/`takeaway` 无图时退化为纯色背景 + 居中大字
- 无 `slides` 数据时自动从 `points`/`quote`/`summary` 生成等分 slides

### 速览场景

速览引导（"快速速览" 或 "更多值得关注" 文字）+ 速览卡片逐张切换。

每张卡片包含：序号、标题、一句话摘要、来源。

**模板适配**:
- Daily: "快速速览"，5-7 张卡片，每张 15-20 秒
- Weekly: "更多值得关注"，3-5 张卡片，每张 15-20 秒
- Article: 无速览段

**节奏要求**: 速览卡片节奏紧凑均匀，每张时长一致，不展开讨论。

### 品牌结尾（3-5 秒）

显示品牌名和标语。仅淡入，无淡出。

## 动效规范

### 基础动效

- **淡入为主**: opacity 0→1 + translateY 10px→0
- **缓动函数**: spring-based 或 ease-out
- **时长**: 0.4-0.8s，不超过 1s
- **Slide 间过渡**: 0.8s 交叉淡入淡出（fade），不用其他转场

### 节奏辅助动效

- **关键词**: 标签依次淡入（每个间隔 6 帧 = 0.2s）
- **速览卡片**: 均匀切换，无额外装饰动效
- **精讲 slides**: 交叉淡入淡出，过渡期间音频不中断

### 禁止的动效

- 翻页、旋转、弹性、缩放等花哨转场
- 文字逐字出现（打字机效果）
- 元素从屏幕外飞入
- 任何持续超过 1s 的过渡

## 图片处理

### 尺寸与裁剪

- 统一处理为 16:9 比例，使用 `object-fit: cover`
- 建议渲染分辨率: 图片区域 1152×648

### 质量要求

- 最小宽度: 800px
- 格式: JPG/PNG/WebP
- 文件大小: ≥10KB

### AI 生成图风格

```
Minimalist tech illustration, flat design, muted color palette
(ink blue #1a365d, cream #fefdfb, gray tones), clean lines,
no text overlays, 16:9 aspect ratio.
```

同一期所有生成图风格一致。

## 观看体验核心检查

视频渲染前对照检查：

- [ ] 没有连续超过 20 秒停留在同一画面
- [ ] slides 切换时配音有自然过渡（不是句子说到一半突然换画面）
- [ ] 高密度 `point` slides 之间有 `quote` 或 `problem` 做呼吸
- [ ] 速览段节奏均匀，不忽快忽慢
- [ ] 开场 10 秒内有视觉吸引力（不是纯文字空白）
- [ ] 图片质量统一，没有明显低质图混入
- [ ] 品牌开场和结尾简洁，不占过多时长

---

## 设计禁忌

- **不要** 使用蓝紫渐变科技感
- **不要** 添加发光、粒子、毛玻璃效果
- **不要** 给卡片加投影
- **不要** 使用超过两种强调色（墨蓝 + 琥珀）
- **不要** 画面塞满信息（每张 slide 最多一个核心观点）
- **不要** 文字动画过快或过花哨
- **不要** 速览区使用和精讲一样复杂的布局
- **不要** 连续 4+ 张高信息密度 slides（会让观众疲劳）

## 平台适配

### B站

- 默认横版 16:9
- 封面图: 从视频关键帧或精讲封面图中选取，叠加标题文字
- 前 3 秒必须有视觉吸引力，B站用户快速滑动，开场不能太慢
- 时间戳导航写入描述

### YouTube

- 同 B站，横版 16:9
- 缩略图单独生成，大字 + 对比色 + 简洁画面
- 英文字幕可选

### 微信视频号

- 可选竖版 9:16，Remotion 渲染参数 `1080×1920`
- 竖版布局调整: 图片占上半部，文字占下半部
- 时长建议 3-5 分钟（视频号偏好短内容）
