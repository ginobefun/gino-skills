# Daily Digest - 每日技术早报

_结合 BestBlogs 设计思路 + 免费数据源 + OpenClaw 定时推送_

---

## 触发短语

- "每日早报" / "今日早报" / "daily digest"
- "今天有什么技术新闻" / "AI 动态"
- "生成早报" / "技术简报"

---

## 数据源（免费，无需 API）

| 分类 | 数据源 | 抓取方式 |
|------|--------|----------|
| AI/ML | Hacker News AI 标签 | browser snapshot |
| 编程技术 | Hacker News | browser snapshot |
| 商业科技 | Hacker News | browser snapshot |
| 国内动态 | 微博热搜 + 知乎热榜 | web_fetch |
| 大厂动态 | 各公司博客 RSS | web_fetch |

---

## 筛选逻辑（4 层优先级）

### 优先级 1: 头部厂商重大动态
以下厂商的新模型、新产品、新政策**自动入选**：
- **国际**: OpenAI, Anthropic, Google/Gemini, Meta AI, xAI, Microsoft
- **国内**: 阿里巴巴/通义，腾讯，DeepSeek，月之暗面/Kimi，智谱/GLM

### 优先级 2: AI Coding 专题
关于以下工具的内容**加权**：
- Claude Code, Codex, Cursor, Windsurf, GitHub Copilot
- AI 辅助编程、Agent 开发、MCP 协议

### 优先级 3: 高优来源加权
- **国际**: Lenny's Podcast, Y Combinator, The Pragmatic Engineer, a16z, Simon Willison
- **国内**: 宝玉，张小珺，腾讯科技，极客公园

### 优先级 4: 内容质量
- Hacker News 评分（≥100 分优先）
- 评论数（≥50 条优先）
- 深度分析 > 短资讯

## 输出格式

### 1. 纯文本版（飞书/微信）

```
📰 技术早报 | YYYY-MM-DD

🔥 头条
[1] 标题
2-3 句摘要，概括核心信息和为什么值得关注
来源：XXX | HN 评分：96
链接

⚡ AI 动态
[2] 标题
一句话摘要
来源：XXX | HN 评分：93
链接

[3] 标题
...

💡 深度阅读
[标题 + 链接]

🏷️ 关键词：AI Coding / Claude / DeepSeek

---
Generated by Daily Digest
```

### 2. 杂志风 HTML

参考 BestBlogs 设计：
- 墨蓝 `#1a365d` + 琥珀 `#d97706`
- 衬线体标题 + 无衬线正文
- 纸质噪点纹理
- Top 1 琥珀色左边线强调
- max-width 640px 居中

### 3. 分享卡片（可选）

用 `baoyu-cover-image` 生成封面图

## 工作流

```
阶段 1: 数据抓取（并行）
  - Hacker News（browser snapshot）
  - 微博热搜（web_fetch）
  - 知乎热榜（web_fetch）

阶段 2: AI 筛选 Top 10
  - 合并去重
  - 按 4 层优先级排序
  - 记录入选理由

阶段 3: 生成输出
  - 纯文本版
  - HTML 版（可选）
  - 封面图（可选）

阶段 4: 保存 + 推送
  - 保存到 contents/daily-digest/YYYY-MM-DD/
  - 飞书推送（如果配置了）
```

## 定时任务配置

在 `HEARTBEAT.md` 中添加：

```markdown
### 每日新闻摘要（每天 9:00-12:00）
- **动作**: 调用 daily-digest skill
- **输出**: 纯文本 + HTML
- **推送**: 飞书消息
```

---

## 错误处理

- 数据源失败 → 降级用其他源
- HN 无法访问 → 用 Reddit r/technology
- 所有源失败 → 告知用户，明天重试
