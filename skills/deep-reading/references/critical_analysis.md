# 批判性分析框架：批判性思维 + 逆向思维

## 一、批判性思维

### 论证质量评估

#### 证据评估矩阵

| 证据类型 | 可信度 | 警惕信号 |
|----------|--------|----------|
| 同行评审研究 | 高 | 样本量、利益冲突 |
| 原始数据 | 高 | 采集方法、偏差 |
| 专家共识 | 中高 | 领域共识 vs 个人观点 |
| 案例研究 | 中 | 选择偏差、可推广性 |
| 轶事/故事 | 低 | 不具代表性 |
| "研究表明…"（无引用） | 极低 | 模糊、不可验证 |

#### 逻辑谬误清单

**因果谬误：**
- 后此谬误（A在B前 ≠ A导致B）
- 相关非因果（相关性 ≠ 因果性）
- 过度简化（单因素解释复杂现象）

**证据谬误：**
- 挑樱桃（选择性使用证据）
- 幸存者偏差（只看到成功案例）
- 以偏概全（小样本 → 大结论）

**修辞谬误：**
- 人身攻击（攻击人而非论点）
- 诉诸权威（权威非该领域专家）
- 稻草人（歪曲对方立场再攻击）
- 滑坡谬误（无依据的极端推演）
- 虚假两难（明明有更多选项却只给两个）

**统计谬误：**
- 基础概率忽视（忽略先验概率）
- 绝对风险 vs 相对风险混淆
- 误导性平均值（均值掩盖分布）

### 批判性追问协议

**层级 1 — 理解：**
- 核心主张是什么？
- 支撑证据是什么？
- 关键假设是什么？

**层级 2 — 分析：**
- 证据是否足以支撑主张？
- 是否存在逻辑跳跃？
- 论证中缺少了什么？

**层级 3 — 评价：**
- 整体论证有多强？
- 什么能加强/削弱它？
- 有哪些替代解释？

**层级 4 — 综合：**
- 与其他已知知识如何契合？
- 即使有瑕疵，作者可能正确的地方在哪？
- 最善意的解读是什么？

### 来源可信度评估

- **作者背景**: 是否有该领域相关专长？是否存在利益冲突？
- **发表背景**: 是否经过同行评审？是原始来源还是二手解读？
- **动机分析**: 目标是告知、说服还是销售？谁从此主张中受益？

### 钢人论证法

在批评之前，先构建该论证**最强版本**：
1. 善意填补逻辑缺口
2. 补充最有利的支撑证据
3. 预先回应明显的反驳
4. 然后评估这个最强版本

这确保了评价的公正性，避免攻击稻草人。

## 二、逆向思维

### 核心原则

"反过来想，总是反过来想。" —— 卡尔·雅可比

不问"如何成功"，而问"如何确保失败"，然后避开那些失败路径。

### 三种逆向模式

#### 1. 目标逆向

正向：如何实现目标 X？
逆向：如何保证无法实现 X？

```
正向目标: [文章的核心建议]
逆向追问:
1. 什么行为保证失败？
2. 哪些因素会彻底破坏计划？
3. 最糟糕的结果是什么？
洞察: 必须避免的行为 + 关键风险点 + 防御策略
```

#### 2. 假设逆向

正向：若假设 A 成立，则结论 B 有效
逆向：若假设 A 不成立，会怎样？

```
文章核心假设:
1. [假设 1] → 反转后结论如何变化？什么条件下会失败？
2. [假设 2] → 是否有反例？
发现的脆弱性: [列出]
```

#### 3. 因果逆向

正向：A 导致 B（因→果）
逆向：要得到 B，什么必须为真？（果→因）

```
期望结果: [文章承诺的效果]
反向推导:
- 实现它需要什么前提条件？
- 文章是否提及了所有前提？
- 缺失的关键因素: [重要发现]
```

### 逆向思维工具箱

#### 事前验尸（Pre-Mortem）

假设一年后计划完全失败：
1. [最可能的失败原因 1]
2. [最可能的失败原因 2]
3. [最可能的失败原因 3]

基于此，当前应采取的预防措施：[列出]

#### 不做清单（比待办更重要）

基于逆向分析，明确要避免的行为：
1. ❌ [导致失败的行为 1]
2. ❌ [导致失败的行为 2]

停止信号：当 [X 信号] 出现时，立即停止

#### 魔鬼代言人

为相反观点辩护：
- 主张: [反面立场]
- 证据: [支持反面的证据]
- 场景: [什么时候反面更好]
- 平衡结论: [综合判断]

### 使用逆向思维的注意事项

- **不是悲观主义**: 目标是增强方案的稳健性，而非否定一切
- **注意概率**: 低概率风险和高概率风险应区别对待（风险 = 概率 × 影响）
- **避免分析瘫痪**: 识别风险 → 设计缓解 → 行动，不要被风险困住

### 与批判性思维的配合

批判性思维找到论证的漏洞，逆向思维追问"如果这些漏洞导致失败会怎样"。

批判性思维是 X 光片，逆向思维是压力测试。
