# 素材清单格式规范 (materials.md)

`materials.md` 使用以下结构化格式，供 content-synthesizer 直接解析：

```markdown
# 阅读素材 | YYYY-MM-DD

## 推文灵感

### 1
- source: [文章标题]
- source_url: [URL]
- core_point: [核心观点一句话]
- my_thought: [个人思考]
- type: tweet

### 2
...

## 博客素材

### 1
- topic: [主题]
- sources: [来源文章标题列表]
- source_urls: [URL 列表]
- angle: [个人切入角度]
- key_quotes: [可引用的金句]
- type: blog

## 金句

1. "[金句]" — [出处]
2. ...

## 核心收获

1. [洞察 1]
2. [洞察 2]
3. [洞察 3]
```
